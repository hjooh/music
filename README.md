# Setup Instructions

### YouTube Music User

1. Download "Pear Desktop" (YT Music desktop app):
    https://github.com/pear-devs/pear-desktop?tab=readme-ov-file#windows

2. Enable Discord Rich Presence (Plugins -> Discord Rich Presence -> SELECT Enable & Auto Reconnect)

3. Enable Activity Status on Discord (Settings -> Activity Privacy -> Share my activity)

4. Add YouTube Music to Registered Games on Discord (Settings -> Registered Games -> Add it! -> ADD)

### Spotify User

1. Set up a Spotify client ID and client secret through the Spotify Developer Portal

2. Update your .env file with the following:
    - Spotify Client ID
    - Spotify Client Secret
    - Spotify Redirect Link
    - Discord Bot Token
  
3. Run python script

**NOTE:** Can only be run from the Spotify user's side currently (can track YT Music -> Spotify, but not the other way around - WIP!)
